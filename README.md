# EduRegister
EduRegister — English &amp; Marathi-language school management web app
# 🏫 EduRegister — ऑनलाइन मराठी शालेय अभिलेख प्रणाली

A comprehensive,  school management system built as a single HTML file
with vanilla JavaScript, Mysql and Nodejs. Designed for Maharashtra primary and secondary
schools with full Marathi language support.

## ✨ Features

- 👨‍🎓 Student General Register — GR numbers, roll numbers, photos, caste/category
- 📋 Lecture-wise & Daily Attendance — monthly reports, calendar view
- 📊 Results & Grade Cards — auto percentage, grade calculation, printable marksheets
- 📜 Certificates — Bonafide, School Leaving, Transfer Certificate generation
- 🪪 Student ID Cards — front & back, printable
- 👩‍🏫 Teachers Management — staff profiles, subjects, service records
- 🕐 Class Timetable — editable weekly timetable per class/division
- 💰 Fees Management — collection tracking, dues, receipts
- 📚 Library Management — book catalog, issue & return records
- 🚌 Transport Management — bus routes, vehicles, student allocation
- 📢 Notice Board — school announcements and circulars
- 🏖️ Holidays & School Calendar — event and holiday management
- 💬 WhatsApp Notifications — send attendance, results, fee reminders to parents
- 🏫 School Profile & Settings — UDISE, board, management details
- 📒 Kird-Khatavani — traditional Marathi school register format
- 🍱 MDM (Mid-Day Meal) Register — rice stock, beneficiary, taste register
- 🔐 Role-based Login — Admin, Head Master, Lipik, Teachers (37 accounts)
- 🌐 Bilingual UI — English & Marathi with Google Transliteration support

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (no frameworks)
- **Storage:** Mysql,PHP MyAdmin,Node.js
- **Fonts:** Google Fonts — Playfair Display, Noto Sans Devanagari, DM Sans
- **Print:** Built-in print stylesheets for all documents
- **Zero dependencies:** No npm, no build step, no server needed
